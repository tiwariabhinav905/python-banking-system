# Python Banking System

A simple console-based banking system built using Python.  
This program allows users to check their balance, deposit money, withdraw money, and exit the program.

## Features
- Check account balance
- Deposit money
- Withdraw money
- Prevent negative deposits or withdrawals
- Prevent withdrawing more than available balance

## Technologies Used
- Python

## How to Run

1. Clone the repository
2. Open the file in Python
3. Run the program

python banking_program.py
